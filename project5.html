ZSL Human Disease Classifier

Built a Zero-Shot Learning (ZSL) classifier for human diseases, enabling the model to identify diseases based on unseen data. The model uses BERT embeddings for textual input and a custom CNN for image processing to provide accurate diagnoses.

Technologies Used:

  • Zero-Shot Learning (ZSL)
  • BERT Embeddings
  • Custom CNN Architecture
  • Medical Image Classification
